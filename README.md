## ✨ Features

- 🔥 Full-screen hero with animated particle canvas & shooting stars
- 🪟 Glass morphism cards and UI elements throughout
- 📜 Scroll-triggered reveal animations (Intersection Observer)
- 🃏 3D tilt effect on anime universe cards
- 🎯 Custom cursor with magnetic ring
- 🌊 Hero parallax on scroll

- 🎨 Anime-themed sections: Naruto, Demon Slayer, One Piece, AOT, DBZ, JJK
- 📱 Fully responsive (mobile, tablet, desktop)
- 🚀 Zero dependencies — pure HTML, CSS, JavaScript

## 🚀 Deploy to GitHub Pages (3 steps)

### Step 1 — Create Repository
1. Go to [github.com/new](https://github.com/new)
2. Name it exactly: `di-wings` *(or any name you like)*
3. Set it to **Public**
4. Click **Create repository**

### Step 2 — Upload Files
Upload both files to the repository root:
- `index.html`
- `README.md`

*(Drag & drop them directly in the GitHub web interface)*

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select `Deploy from a branch`
3. Choose branch: `main` / folder: `/ (root)`
4. Click **Save**

⏳ Wait ~60 seconds, then your site is live at:
```
https://YOUR_USERNAME.github.io/di-wings/
```
